# Todo Application

Простое веб-приложение для управления задачами, построенное на FastAPI.

## Описание

Это приложение предоставляет REST API и веб-интерфейс для управления списком задач (todos). Данные хранятся в памяти и обновляются при каждом перезапуске приложения.

## Функциональность

- Создание, чтение, обновление и удаление задач
- Веб-интерфейс для управления задачами
- REST API для програмmatic доступа
- Поддержка JSON и form-data для создания задач

## Установка и запуск

### Предварительные требования

- Python 3.7+
- pip (менеджер пакетов Python)

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск приложения

```bash
uvicorn api:app --host=0.0.0.0 --port 8000 --reload
```

Приложение будет доступно по адресу: `http://localhost:8000`

**Веб-интерфейс для работы с задачами доступен по адресу:** `http://localhost:8000/todo`

## API Endpoints

### Основные endpoints

- `GET /` - Приветственное сообщение
- `GET /todos` - Получить все задачи
- `GET /todos/{id}` - Получить задачу по ID
- `POST /todos` - Создать задачу (JSON)
- `POST /todos/form` - Создать задачу (форма)
- `PUT /todos/{id}` - Обновить задачу
- `DELETE /todos/{id}` - Удалить задачу

### Веб-интерфейс

- `GET /todo` - Просмотр всех задач
- `POST /todo` - Добавить новую задачу
- `GET /todo/{id}` - Просмотр конкретной задачи
- `PUT /todo/{id}` - Обновить задачу
- `DELETE /todo/{id}` - Удалить задачу
- `DELETE /todo` - Удалить все задачи

## Использование

### Через веб-интерфейс

1. Откройте `http://localhost:8000/todo` в браузере
2. Используйте форму для добавления новых задач
3. Нажмите на задачу для просмотра деталей
4. Используйте API endpoints для управления задачами

### Через API

Примеры запросов с использованием curl:

```bash
# Получить все задачи
curl -X GET "http://localhost:8000/todos"

# Создать задачу
curl -X POST "http://localhost:8000/todos" -H "Content-Type: application/json" -d '{"item": "Новая задача"}'

# Обновить задачу
curl -X PUT "http://localhost:8000/todos/1" -H "Content-Type: application/json" -d '{"item": "Обновленная задача"}'

# Удалить задачу
curl -X DELETE "http://localhost:8000/todos/1"
```

## Технологии

- **FastAPI** - современный веб-фреймворк для построения API
- **Pydantic** - валидация данных и сериализация
- **Jinja2** - шаблонизатор для HTML
- **Uvicorn** - ASGI сервер
- **Bootstrap** - CSS фреймворк для стилизации

## Примечания

- Данные хранятся в памяти и не сохраняются между перезапусками приложения
- При запуске приложения автоматически создаются тестовые задачи
- Для production использования рекомендуется добавить постоянное хранилище данных
